5 HTML tags I learned

<header>
The Header element.
This element represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name, and other elements.
<P>
The Paragraph element.
For adding a paragraph of text we can use The browser will add a space before and after the content of the paragraph.
<strong>
The Strong Importance element.
Import.This element indicates that its contents have strong importance, seriousness, or urgency. Browsers typically render the contents in bold type.
<img>
The Image Embed element.
This element embeds an image into the document.
<body>
The Document Body element.
This element represents the content of an HTML document. There can be only one "body"element in a document.
